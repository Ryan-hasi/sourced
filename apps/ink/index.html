Corroboration as a primitive

Confirmed by sources.
Never “true.”

Given a stream of claims from many origins, Sourced tells you how many independent sources corroborate each one, since when — and hands you the receipts. One small function. No truth oracle attached.

The missing layer

Every feed, every aggregator, every “is this real?” moment re-solves the same problem badly or not at all: a thousand items, each equally loud, no signal for which ones the world actually agrees happened. The reader is left to eyeball logos and guess.

The missing layer is the oldest heuristic in journalism — get a second source — computed continuously over a live stream, honestly. Sourced is that layer. It answers exactly one question: how broadly is this corroborated? And it answers it with receipts, because “six independent outlets report this, first seen 17:04” is a checkable fact about the world’s reporting — not a verdict handed down.

One call

import { assess } from "@sourced/core";

const verdicts = await assess([
  { id: "a", title: "Central bank raises rates", origin: "reuters", publishedAt: "2026-01-01T12:00Z" },
  { id: "b", title: "Central bank raises rates", origin: "bbc",     publishedAt: "2026-01-01T12:04Z" },
]);

// → { corroboration: 2,                  // distinct INDEPENDENT origins
//     corroboratingSources: ["reuters"],  // the receipts
//     firstSeenAt: "2026-01-01T12:…",     // event memory
//     signal: "breaking" }                // confirmed | breaking | developing | null

Zero dependencies. Storage, clustering, clock and thresholds are injected — the primitive only counts. How you group “same event” (LLM, regex, embeddings, an upstream editor) stays out of the block.

The honesty guarantees

Invariants, not preferences — and they ship as an executable adversarial test suite, not as promises. Watch them pass, live.

#Guarantee
G1Undercount, never overcount. Matching is deliberately conservative. A false “confirmed” is a reputation balloon that pops — Sourced always errs toward saying less.
G2Never “true,” only “corroborated.” Truth is not computable from headlines; breadth of independent reporting is.
G3Independence is by origin, not by article. Ten syndicated copies of one wire story count as one.
G4Every verdict carries its receipts — who corroborated, since when. Falsifiable by design.
G5Single sources stay bare. No badge, no signal. Silence protects credibility.
G6Signal rides the reliable clock. Urgency comes from upstream publish time — a matching error can never invent “breaking.”
G7Fail open, never break. Dead storage, garbage input → claims pass through unlabeled. Degrading is safe; crashing is not.
