Confirmed by sources.

The corroboration engine: how many independent sources confirm a claim — since when, and who. The missing trust layer for news, feeds and AI.

The problem

Everything is equally loud.

FEEDS
A thousand headlines, no signal for what the world actually agrees happened. Readers eyeball logos and guess.
AI
Models assert answers with confidence — grounding tools collapse everything to a true/false score nobody can check.
TRUST
Every “verified” badge on the internet is a flag someone put there. Nothing behind it.
The insight

Truth isn’t computable.
Corroboration is.

“Six independent outlets report this, first seen 17:04” is a checkable fact — not a verdict handed down. It’s journalism’s oldest rule — get a second source — computed continuously, honestly, over a live stream.

The primitive

One small function.

const verdicts = await assess(claims)

// per claim:
{ corroboration: 4,                       // distinct INDEPENDENT sources
  corroboratingSources: ["reuters", "bbc", "ap"],  // the receipts
  firstSeenAt: "2026-07-10T17:04Z",       // event memory
  signal: "confirmed" }                   // never "true"

Zero dependencies. Storage, clustering and thresholds are injected — it runs on news, outages, OSINT, sensors, moderation queues.

Why it can be trusted

Honesty you can execute.

UNDERCOUNT
It would rather miss a corroboration than invent one. A false “confirmed” is a reputation balloon that pops.
INDEPENDENT
Ten copies of one wire story count as one. Origins, not articles.
RECEIPTS
No count without who and since when. Single sources stay unlabeled.
PROVEN
All of it ships as an adversarial test suite anyone can run, live — not as promises.
