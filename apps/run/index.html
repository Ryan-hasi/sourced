sourced.run — the playground

Paste claims.
Get verdicts.

The live primitive. Each claim is { id, title, origin, publishedAt } — same titles from different origins corroborate each other; copies from the same origin collapse to one (G3). Single sources stay bare (G5).

Use it as an API

curl -s https://sourced.run/api/assess \
  -H "content-type: application/json" \
  -d '{"claims":[
    {"id":"a","title":"Major dam breach reported upstream","origin":"reuters","publishedAt":"2026-07-10T21:00:00Z"},
    {"id":"b","title":"Major dam breach reported upstream","origin":"bbc","publishedAt":"2026-07-10T21:04:00Z"}
  ]}'

Stateless: corroboration is computed within the batch you send. 200 claims / 200 kB per call. Optional config can TIGHTEN or relax thresholds — the defaults are the honest contract. Spec & guarantees: sourced.ink.
