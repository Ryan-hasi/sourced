# Sourced trademark policy

The code in this repository is MIT-licensed (see LICENSE). The **"Sourced"
name** and the **Sourced mark (the red dot lockup)** are not part of that
license. They identify the original project and its guarantees.

## You may, without asking

- State facts: "built on Sourced", "uses @sourcedhq/core", "compatible with
  the Sourced spec", with links to https://sourced.ink.
- Fork, modify and ship the code (MIT) — under a different name.
- Write about Sourced, screenshot it, benchmark it.

## You may not

- Name a fork, derivative, product or service "Sourced" or something
  confusingly similar, or use the mark for it.
- Imply endorsement by or affiliation with the Sourced project without
  written permission.
- Call an engine **"Sourced-conformant"** unless it passes the current
  @sourcedhq/conformance suite in full, unmodified. The suite is the sole
  yardstick for that claim.

## The mark program

A "✓ Sourced" trust mark for products whose corroboration engine passes the
conformance suite is being established; its terms will be published here.
